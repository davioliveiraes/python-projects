# Agenda de Contatos

Sistema de gerenciamento de contatos via terminal com funcionalidades completas de CRUD e sistema de favoritos.

## Finalidade

Este projeto foi desenvolvido como exercício prático para consolidar os fundamentos de Python, abordando:

- **Fundamentos de Python**: estruturas de dados (listas e dicionários), controle de fluxo com `match/case`, funções, manipulação de strings e entrada/saída de dados
- **CRUD Completo**: implementação das operações de Create, Read, Update e Delete para gerenciamento de contatos
- **Lógica de Programação**: ordenação customizada, mapeamento de índices e validação de dados

## Tecnologias

- Python 3.13

## Como Executar

```bash
git clone https://github.com/davioliveiraes/python-projects.git
cd python-projects/simple_agenda
python main.py
```
